# ixigo-hotels
---------------------

Matches similar hotels using GPS coordinates and levhensteins' distance in name of hotels.
Sample data collected from Ibibo, Make my Trip, Expedia etc.

### Instructions

1. Goto `dist` folder
2. run 
```
java -jar ixigo-runnable.jar
```
